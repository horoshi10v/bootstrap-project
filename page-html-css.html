Bootstrap

Bootstrap – CSS-фреймворк для створення крос-браузерних та стандартизованих сайтів з адаптивною версткою.

Адаптивність – його головна перевага.

Фреймворк використовують frontend-, fullstack-розробники. Для роботи з інструментом потрібні мінімальні знання верстки, тому він підходить новачкам.

В цілому, це набір файлів. Після їх підключення до сторінки для верстки стане доступна велика кількість готових компонентів та класів. Вони дозволяють швидко та якісно створювати адаптивний дизайн сайту.

Класи в Bootstrap поділяються на 3 великі групи:

  • для створення сітки – адаптивного макета сторінки;
  • для стилізації контенту – тексту, зображень, коду, таблиць та іншої інформації;
  • службові – для вирішення популярних допоміжних завдань, таких як відображення та приховування елементів, вирівнювання тексту на сторінці, налаштування кольору фону, відступів тощо.

Він дотримується принципу "mobile-first", що передбачає початкову розробку сайту для невеликого екрану мобільного телефону, потім контент розміщується згідно з роздільною здатністю планшета і, нарешті, верстальник працює над версією для ноутбука та комп'ютера. Завдяки такому підходу користувач може переглядати сайт на будь-яких пристроях, не відчуваючи найменшої незручності.

Приклад коду Bootstrap, створення сітки:

    <div class="container">
         <div class="row">
             <div class="col-4">
                One of three columns
             </div>
            <div class="col-4">
                One of three columns
            </div>
            <div class="col-4">
                One of three columns
            </div>
        </div>
    </div>

Foundation

Цей фреймворк – один із популярних у сегменті front-end-фреймворків. Він також має назву ZURB Foundation, тому що ZURB – це назва дизайнерської фірми, яка і створила фреймворк.

Фреймворк складається із CSS-файлів. Він був розроблений для того, щоб дозволити швидко збирати прототипи та формувати код для сучасних сайтів та додатків, які будуть чудово виглядати на будь-яких пристроях.

Foundation був першим фреймворком, який прийняв підхід "mobile-first".

Особливості фреймворку:

  • Гнучкий макет, що підтримує порядок джерел, зміщення та подання пристроїв.
  • Численні стилі навігації, включаючи комплексну головну панель, яка підтримує трирівневу навігацію для простих панелей, сайдбарів і додаткових ділянок навігації.
  • Довільний плагін Clearing для виведення на екран адаптивних лайтбоксів
  • Reveal – довільний плагін jQuery, що дозволяє виводити прості модальні вікна для всіх пристроїв.
  • Joyride – кроссбраузерний плагін, що дозволяє створити екскурсію по вашому сайту або плагіну.

Приклад використання коду Foundation, вкладені стовпці:

    <div class="row">
        <div class="small-6 columns">
            <div class="row">
                <div class="small-6 columns">...</div>
                <div class="small-6 columns">...</div>
            </div>
        </div>
    </div>

UIKIT

Фреймворк відрізняється модульною легкою структурою. Є кілька особливостей, які дозволяють йому виділятися натомість сучасних фреймворків. Це markdown – можливість попередньо переглянути сторінку сайту в режимі реального часу. Також можна відзначити синтаксичне підсвічування для HTML.

UIKit – основний фреймворк для створення та управління інтерфейсом користувача (UI). Існує безліч iOs-китів, які створені для допомоги в розробці програм під iPhone, iPad і т.д.

З яких елементів складається UIKit:

  • eлементи фірмового стилю: модульна сітка, типографіка, кольори, інтервали, зображення, візуальні форми, анімація, звук;
  • елементи інтерфейсу: кнопка та групи кнопок, прапорці (check boxes), форми з полями введення (inputs), підказки, іконки, меню, повідомлення, пагінація, поп-апи, алерти, перемикачі, вкладки та хлібні крихти, таблиці, ярлики та лейбли, галереї, прогрес-бари.
